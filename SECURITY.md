# Security Policy

## Supported Versions

Only the latest version of SqliteClone is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by opening a GitHub Issue or contacting the maintainers directly.

We appreciate your help in keeping this project secure!
